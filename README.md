# 🧠 ZenFit – Self-Hosted Habit & Lifestyle Tracker

A self-hosted, all-in-one habit tracking and lifestyle management app built for personal use.

This project started as a simple alternative to paid apps like Nutriscan, but evolved into a multi-system tracker combining habits, nutrition, workouts, study, and analytics — all in one place.

---

## 🚀 Overview

ZenFit is a fully client-side web app that works as:

* A **habit tracker**
* A **nutrition tracker**
* A **productivity system**
* A **gamified self-improvement tool**

It runs entirely in the browser and stores all data locally — no login, no backend, no tracking.

---

## ✨ Features

### 🧩 Core System

* Self-hosted (via GitHub Pages)
* Works as a web app + Android app (via AppsGeyser)
* Fully offline-capable after load
* No login or authentication required
* All data stored locally (browser storage)

This app is **not intended for medical or scientific accuracy**.
